Om Meg

Jeg er en engasjert dataingeniør med sterk interesse for utvikling, DevOps og automatisering. Jeg arbeider aktivt med moderne utviklingspraksis som CI/CD, GitHub Actions og Azure Pipelines, og har erfaring med både backend- og frontend-teknologier. Jeg er målrettet, fleksibel og jobber godt både selvstendig og i tverrfaglige team. Jeg trives med å utvikle robuste løsninger, optimalisere arbeidsflyter og skape effektive utviklingsprosesser.
